## 🛠 Installation
To run this project locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/holysinner22/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your browser.
